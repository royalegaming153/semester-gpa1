# GPA & CGPA Calculator

## Description
This is a simple web-based GPA and CGPA calculator built using HTML, CSS, and JavaScript. It helps students calculate their GPA (Grade Point Average) and CGPA (Cumulative Grade Point Average) easily and quickly.

## Features
- Calculate GPA for a semester  
- Calculate CGPA for multiple semesters  
- Simple and user-friendly interface  
- Fast and accurate results  
- Responsive design for mobile and desktop

## How to Use
1. Open the project folder  
2. Run the `index.html` file in your browser  
3. Enter your marks or grades  
4. Click on calculate to get GPA/CGPA results  

## Technologies Used
- HTML  
- CSS  
- JavaScript  

## Purpose
This project is created for students to easily calculate GPA and CGPA and to practice front-end development skills.
